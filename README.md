# Makefile using
## To compile and execute a part of project
### client, board or server
Go into the directory :
```console
cd <directory> 
```
Compile :
```console
make 
```
Execute :
```console
make execute
```

## To compile and execute the whole project
#### It will be changed to split in different bash
Compile :
```console
make 
```
Execute :
```console
make execute
```
