# Makefile using
## To compile and execute a part of project
### client, board or server
Go into the directory :__
```console
cd <directory> 
```
Compile :__
```console
make 
```
Execute :__
```console
make execute
```

## To compile and execute the whole project
#### It will be changed to split in different bash
Compile :__
```console
make 
```
Execute :__
```console
make execute
```
